# Ciencia de Datos

---

## Descripción general

El contenido del repositorio está orientado al desarrollo de habilidades prácticas en las siguientes áreas:

- Manipulación y análisis de datos  
- Conexión a bases de datos  
- Procesamiento de información estructurada y no estructurada  
- Análisis exploratorio de datos  
- Procesamiento de lenguaje natural (NLP)  

Todas estas actividades se desarrollan utilizando Python como lenguaje principal.

---

## Fuentes de datos

Se emplean diversos conjuntos de datos para la aplicación de técnicas de análisis, entre los que destacan:

- Datos de películas (IMDB)  
- Datos de texto para análisis de sentimientos  

Estos datasets permiten abordar tanto el análisis numérico como el procesamiento de información textual.

---

## Conexión a bases de datos (MongoDB)

Se incorpora el uso de MongoDB como sistema de base de datos NoSQL, empleando la librería `pymongo` para:

- Establecer conexión con un clúster  
- Verificar la conexión mediante comandos como `ping`  
- Preparar el entorno para el almacenamiento y consulta de datos  

Este enfoque permite trabajar con datos no estructurados y escalables.

---

## Análisis de datos con Pandas

Se utiliza la librería `pandas` para el manejo y análisis de datos, incluyendo:

- Carga de datasets en formato `.csv`  
- Exploración inicial mediante métodos como `.head()`  
- Limpieza y transformación de datos  
- Preparación de la información para análisis posteriores
