# Data Manipulation

En esta sección se aborda la manipulación de datos utilizando Python, con un enfoque práctico orientado a la limpieza, transformación y normalización de datasets mediante la librería `pandas`, apoyándose también en `NumPy` para la generación de datos.

Se emplean tanto datos sintéticos como datos reales con el fin de comprender el flujo completo de preparación de la información. Por un lado, se generan conjuntos de datos controlados a partir de distribuciones estadísticas, lo que permite analizar su estructura y comportamiento. Por otro lado, se utilizan archivos en formato `.csv`, como el dataset *Students Social Media Addiction*, para trabajar con información del mundo real.

El proceso incluye tareas fundamentales como:

- Limpieza de datos (manejo de valores nulos, corrección de formatos y eliminación de duplicados)  
- Transformación de variables  
- Normalización de datos  

Estas técnicas son esenciales para garantizar la calidad de la información y prepararla adecuadamente para análisis posteriores o la construcción de modelos de *machine learning*.

Asimismo, se realizan operaciones de exploración y manipulación, tales como:

- Filtrado de datos  
- Creación de nuevas variables  
- Reorganización de la información  

Lo anterior permite obtener conjuntos de datos más estructurados, consistentes y adecuados para su análisis.

En conjunto, esta sección proporciona una base sólida en el preprocesamiento de datos, etapa fundamental dentro del flujo de trabajo en ciencia de datos.
