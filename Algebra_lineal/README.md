## Álgebra Lineal

En esta sección se introduce el uso de Python como herramienta para el análisis matemático. Se comienza con un repaso de los tipos de datos básicos, como listas, números enteros y cadenas de texto.

Posteriormente, se estudia el concepto de vectores tanto desde un enfoque teórico como práctico, utilizando librerías como NumPy y Matplotlib para su manipulación y visualización.

Entre los temas que se desarrollan se incluyen:

- Vectores en 2D y 3D
- Norma de un vector
- Suma de vectores
- Multiplicación por un escalar
- Vector cero y vector unitario
- Normalización de vectores
- Producto punto
- Cálculo del ángulo entre vectores
- Proyección de un vector sobre un eje

Mediante el uso de Matplotlib, se generan representaciones gráficas que facilitan la comprensión del comportamiento de los vectores en el espacio.

Además, se introduce la librería SymPy, que permite realizar cálculo simbólico en Python. Esta herramienta es útil para trabajar con expresiones algebraicas de forma exacta, resolver ecuaciones, calcular derivadas y realizar manipulaciones matemáticas sin recurrir a aproximaciones numéricas.

Finalmente, se aborda el concepto de subespacios fundamentales, proporcionando una base teórica clave para comprender el álgebra lineal y sus diversas aplicaciones.
