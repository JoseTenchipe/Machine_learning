# Streamlit

En esta sección se desarrollan aplicaciones interactivas utilizando `Streamlit`, enfocadas en la visualización y simulación de modelos matemáticos y estadísticos. El objetivo principal es transformar conceptos teóricos en herramientas dinámicas que permitan la exploración de datos y procesos en tiempo real.

Se implementa un dashboard basado en programación orientada a objetos (OOP) para el análisis de una cadena de Markov de dos estados (V/C), incorporando visualizaciones interactivas mediante la librería `Altair`. Este enfoque permite representar de manera clara la evolución de los estados y sus probabilidades, evitando la dependencia de otras herramientas como `Plotly`.

Asimismo, se desarrollan aplicaciones orientadas a la simulación de procesos estocásticos, incluyendo:

- Procesos de Poisson homogéneos (PPH)  
- Procesos de Poisson no homogéneos (PPNH)  

Estos modelos permiten representar fenómenos en los que los eventos ocurren de manera aleatoria en el tiempo, ya sea con una tasa constante o variable. Entre sus aplicaciones se encuentran el conteo de eventos físicos, el tráfico web, las llegadas a sistemas de servicio y diversos fenómenos naturales.

Adicionalmente, se integran elementos de análisis financiero, como la construcción y comparación de portafolios bajo el enfoque de Markowitz. Se consideran variantes que incorporan matrices de covarianza ajustadas mediante técnicas de limpieza, como *Random Matrix Theory (RIE)*, así como comparaciones entre resultados *in-sample* y *out-of-sample* utilizando múltiples activos financieros.
