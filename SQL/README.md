# SQL

SQL (Structured Query Language) es un lenguaje especializado utilizado para gestionar y manipular datos almacenados en sistemas de bases de datos relacionales (RDBMS). Constituye una herramienta fundamental en áreas como el análisis de datos, la ciencia de datos y el desarrollo de software.

---

## Ruta de aprendizaje

### 1. Introducción a bases de datos relacionales

Una base de datos relacional organiza la información en múltiples tablas que se encuentran relacionadas entre sí.

Entre sus principales características se destacan:

- Datos estructurados en filas y columnas  
- Cada fila representa un registro único  
- Cada columna define un tipo de dato específico  
- Existencia de relaciones entre tablas mediante claves (keys)  

Este modelo permite realizar consultas complejas mediante la combinación de información proveniente de distintas tablas.

---

### 2. Sistemas de gestión de bases de datos (DBMS)

Un sistema de gestión de bases de datos (DBMS, por sus siglas en inglés) es el software encargado de crear, administrar y manipular bases de datos.

Algunos de los sistemas más utilizados son:

- MySQL  
- PostgreSQL  
- SQLite  
- Oracle Database  
- IBM Db2  

A pesar de la diversidad de sistemas existentes, la mayoría emplea SQL como lenguaje estándar, lo que facilita la transferencia de conocimientos entre distintas tecnologías.

---

### 3. SQL como herramienta en ciencia de datos

SQL desempeña un papel esencial dentro del análisis de datos, ya que permite:

- Consultar grandes volúmenes de información  
- Filtrar datos relevantes  
- Realizar operaciones de agregación (SUM, AVG, COUNT)  
- Combinar tablas mediante operaciones JOIN  
- Preparar datos para su posterior análisis o modelado  

Por estas razones, el dominio de SQL constituye una competencia fundamental en el flujo de trabajo de un científico de datos.
